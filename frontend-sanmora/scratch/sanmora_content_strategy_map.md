# Sanmora.in Content Strategy & SEO/AEO Information Architecture Map

This document establishes the structured sitemap, search intent mapping, metadata, and internal link routing for Sanmora's 15 core service pages, comparison hubs, and resource guides.

---

## 1. Content Architecture Overview

To rank high on Google Search and win citations in AI engines (ChatGPT, Gemini, Perplexity), Sanmora must structure its content using a **Topical Silo (Clustering) Model**. 

*   **Pillar Pages (Core Services):** Broad high-volume terms targeted with comprehensive pages.
*   **Sub-Service Pages (Developer Tech Stack):** Highly technical pages matching developers' search queries.
*   **Comparison Hubs:** Targeting comparison queries with structured tables and balanced recommendations.
*   **Resource Guides:** Target informational long-tail searches (top-of-funnel leads) and serve as sources for links.

```
                    [Services Overview Page]
                                |
       +------------------------+------------------------+
       |                                                 |
[Website Dev Services]                        [Custom Software Services]
       |                                                 |
  +----+----+                                       +----+----+
  |         |                                       |         |
[Next.js] [React]                               [Custom CRM] [Web App Guide]
  |         |                                       |         |
  +----+----+                                       +----+----+
       |                                                 |
       +---------------> [Internal Link Mesh] <----------+
```
